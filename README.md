# gemini-beautify-2

让 Gemini 可以换背景 （AI 辅助制作）

## 📖 简介

这是一个浏览器插件，允许用户自定义 Gemini 的背景图片，提升用户体验。

## 🚀 快速开始

### 安装

#### 方法一：从压缩包安装（开发者模式）

1. **打开浏览器开发者模式**
   - 在浏览器地址栏输入 `chrome://extensions/`
   - 启用左上角的"开发者模式"开关

2. **解压文件**
   - 将插件压缩包解压到本地文件夹

3. **加载扩展**
   - 点击左上角的"加载未打包的扩展应用"按钮
   - 选择刚才解压出来的文件夹

4. **确认安装**
   - 扩展将被添加到浏览器中

<img width="722" height="282" alt="开发者模式加载扩展应用" src="https://github.com/user-attachments/assets/d8733a8c-c64f-4ea7-9dfc-38e9225d766d" />

### 使用方法

1. **选择背景图片**
   - 点击"点击选择图片文件"按钮
   - 从本地选择你喜欢的背景图片

2. **应用背景**
   - 点击"应用背景"按钮
   - 背景将立即生效
